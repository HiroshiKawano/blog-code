## 4. 設定の変更と絵の描きかた
### 座標
- Processing はウィンドウ左上が原点 (x=0, y=0)。
- 横方向はx, 縦はy。

### 線と図形
```
//-> 線を描く
//// sx, sy: 始点、ex, ey: 終点（長さではない）
line(sx, sy, ex, ey);

//-> 四角形
//// x, y: 左上頂点、w, h: 幅と高さ
rect(x, y, w, h);

//-> 円（だ円）
//// x, y: 円の中心点、w, h: 横と縦の直径
//// 横と縦の直径を別の値にするとだ円になる
ellipse(x, y, w, h);

//-> 三角形  
//// 引数で３つの頂点(x*, y*)を指定
triangle(x1, y1, x2, y2, x3, y3);
```


## 5. 色を変更しよう
```
background(128, 128, 128);  // 背景色
noStroke();  // 線なし
stroke(255, 0, 0);  // 線の色
strokeWeight(5);  // 線の太さ
```
