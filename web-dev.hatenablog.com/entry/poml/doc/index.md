[Poml](https://github.com/mamorum/poml) を使うと、[Maven](https://maven.apache.org/index.html) の `pom.xml`（プロジェクト定義ファイル）をシンプルに書くことができます。

Poml は、以下の２つから構成されています。

1. `pom.poml` を書くための構文
2. `pom.poml` を `pom.xml` に変換するツール

`pom.poml` は、Poml ファイルと呼んでいます。
