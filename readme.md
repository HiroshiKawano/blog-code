## はてなブログとの同期
はてなブログに投稿した記事（Markdown）は、`sync.bat` で取得（同期）しています。

`sync.bat` は、記事「[blogsync ではてなブログのエントリをローカルと同期する](http://motemen.hatenablog.com/entry/2014/12/22/blogsync)」のツール `blogsync.exe` を利用しています。
