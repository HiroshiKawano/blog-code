## 概要
[Web系開発メモ（はてなブログ）](http://web-dev.hatenablog.com/)の資源を置いています。

はてなブログの記事（Markdown）は、[blogsync](https://github.com/motemen/blogsync) で取得しています。


### blogsync のコマンド例
```
> blogsync pull web-dev.hatenablog.com
```

※ 記事の削除は反映されないみたいです。
