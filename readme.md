## はてなブログの Markdown について
はてなに投稿した Markdown を、`sync-md.bat` で取得しています。

`sync-md.bat` は、記事「[blogsync ではてなブログのエントリをローカルと同期する](http://motemen.hatenablog.com/entry/2014/12/22/blogsync)」のツール `bin/blogsync.exe` を利用しています。
